⚠ Les coordonnées, seuils de tension et poids sont interprétatifs — construits à partir d'une analyse de la transcription, pas des mesures objectives. Le placement encode un jugement argumenté.

⚠ L'attribution des voix repose sur une reconstruction par le contenu et les prénoms d'adresse. La diarisation automatique ayant échoué sur le cœur du débat, les attributions sont des hypothèses.

⚠ Environ 19 % du débat (≈ 16 min sur 85) sont non enregistrés par refus de participants. Les zones grisées signalent une mesure absente, pas un silence. Toute lecture de tension dans ces fenêtres est indéterminée.
